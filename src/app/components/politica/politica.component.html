O HoverTraduz é uma extensão de navegador que traduz textos no hover e converte moedas em páginas da web, sem substituir o conteúdo original.

1️⃣ Quem somos

Nome do produto: HoverTraduz

Contato de suporte: suporte@jrweb.com.br

2️⃣ Dados que processamos

O HoverTraduz não coleta dados pessoais identificáveis (como nome, e-mail, telefone, CPF, etc.) para venda ou perfilamento.

Para funcionar, a extensão pode processar:

  • Trechos de texto visível da página (para tradução no hover)
  • Valores de preço (para conversão de moeda)
  • Preferências do usuário (idioma, moeda e opções da extensão)
  • Cache técnico local para melhorar desempenho

3️⃣ Como usamos esses dados

Os dados processados são usados exclusivamente para:

  • Mostrar tradução de texto no hover
  • Exibir conversão de moeda ao lado dos preços
  • Salvar configurações escolhidas pelo usuário
  • Reduzir requisições repetidas via cache

4️⃣ Compartilhamento com terceiros

Quando a tradução online estiver habilitada, trechos de texto podem ser enviados a provedores externos de tradução para gerar o resultado.

A extensão também pode consultar provedores de cotação para conversão de moeda.

✅ O HoverTraduz não vende dados de usuários.

5️⃣ Armazenamento e retenção

As preferências e o cache técnico são armazenados localmente no navegador (ex.: armazenamento da extensão).

Esses dados permanecem até serem sobrescritos, limpos pelo navegador ou removidos com a desinstalação da extensão.

6️⃣ Segurança

Adotamos medidas para minimizar o processamento de dados e limitar o uso ao objetivo principal da extensão.

7️⃣ Seus direitos

Você pode:

  • Desativar recursos da extensão
  • Remover a extensão a qualquer momento
  • Limpar os dados locais pelo navegador

8️⃣ Crianças e menores

A extensão não é direcionada especificamente a crianças e não busca coletar dados pessoais de menores.

9️⃣ Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente.

A versão mais recente estará sempre disponível nesta mesma URL.
